# 🎨 Meu Portfólio — Projeto de Estudos HTML & CSS
 
Este projeto é um portfólio pessoal desenvolvido com o objetivo de **praticar e consolidar meus conhecimentos em HTML e CSS**. Aqui você encontra a estrutura, o estilo e as decisões tomadas durante o desenvolvimento.
 
## 📋 Sobre o projeto
 
Este é um projeto de estudos, criado para colocar em prática conceitos fundamentais de desenvolvimento web front-end, como estruturação semântica de páginas, estilização, responsividade e boas práticas de organização de código.
 
## 🎯 Objetivos de aprendizado
 
- Praticar a estruturação semântica de páginas com HTML5
- Aplicar estilos com CSS puro (sem frameworks)
- Criar um layout responsivo (mobile-first)
- Organizar arquivos e pastas de um projeto front-end
- Praticar boas práticas de acessibilidade
## 🛠️ Tecnologias utilizadas
 
- **HTML5** — estruturação do conteúdo
- **CSS3** — estilização e layout
